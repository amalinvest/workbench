# Security Policy

## Supported Versions

Only the latest minor release receives security updates.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a Vulnerability

Please **do not open a public issue** for security vulnerabilities.

Email: `security@getworkbench.dev`

Include:

- A description of the vulnerability
- Steps to reproduce
- The affected version(s)
- Your name / handle for credit (optional)

We will acknowledge receipt within 3 business days and aim to provide a fix or mitigation plan within 14 days for confirmed vulnerabilities.
